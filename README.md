# 🍯 Miel de Abeja Cazador

Sitio web oficial de **Miel de Abeja Cazador**, un emprendimiento costarricense dedicado a la producción y venta de miel de abeja **100% natural, pura y artesanal**.

La miel es producida de forma responsable en apiarios ubicados en la **Zona de Los Santos**, Costa Rica, respetando a las abejas y al entorno natural.

## 🐝 Sobre el proyecto

Este sitio web tiene como objetivo:

- Dar a conocer el emprendimiento **Miel de Abeja Cazador**
- Informar sobre los distintos tipos de miel disponibles
- Mostrar el proceso artesanal de producción
- Facilitar el contacto directo con clientes vía WhatsApp y redes sociales

---

## 🍯 Tipos de miel

El sitio presenta información sobre:

- **Miel de Apis mellifera**  
  Miel tradicional de abeja europea, dulce y cremosa.

- **Miel de Mariola (Tetragonisca angustula)**  
  Miel de abeja nativa sin aguijón, apreciada por sus propiedades medicinales.

- **Miel de Soncuano (Scaptotrigona pectoralis)**  
  Miel artesanal de sabor intenso y alto valor nutricional.
